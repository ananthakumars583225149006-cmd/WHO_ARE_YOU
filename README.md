# 🛡️ WHO ARE YOU? - Cybersecurity Personality Quiz

Discover your true cybersecurity identity! This interactive quiz asks 15 questions about your interests, thinking style, and problem-solving approach to reveal which cybersecurity role suits you best.

## ✨ Live Demo

**[Try the Quiz Here](https://your-username.github.io/who-are-you-cyber-quiz/)**

> ⚠️ Replace `your-username` with your actual GitHub username

## 📸 Screenshot

![WHO ARE YOU? Quiz Screenshot](screenshot.png)

## 🎯 What Makes This Special

This isn't a test of knowledge - it's a **personality discovery journey**. There are no right or wrong answers. Each question presents four options, and each option represents a different cybersecurity domain. Your choices reveal where your true interests lie!

## 🎭 The 6 Cybersecurity Roles

| Role | Emoji | Description |
|------|-------|-------------|
| **Blue Team Analyst** | 🔵 | Defender, monitoring networks, detecting threats, incident response |
| **SOC Analyst** | 🟢 | Security operations, investigating alerts, threat hunting |
| **Red Team Specialist** | 🔴 | Offensive security, penetration testing, ethical hacking |
| **Security Architect** | ⚪ | Designing secure systems, security strategy, building defenses |
| **Cyber Risk Analyst** | 🟡 | Risk assessment, compliance, governance, security frameworks |
| **Digital Forensics Investigator** | 🟣 | Evidence analysis, incident investigation, tracking attackers |

## 📋 How It Works

1. **Enter Your Name** - Personalize your journey
2. **Welcome Page** - "Welcome to Cyber World" with your name
3. **15 Questions** - Each with 4 options, choose what feels right to you
4. **Instant Results** - Discover your cybersecurity role with description

## ❓ Sample Questions

**"When faced with a security alert, what do you do first?"**
- 🔵 Investigate and verify if it's real
- 🔴 Check if it can be exploited
- ⚪ Document for future reference
- 🟡 Look for patterns across alerts

**"What excites you most about cybersecurity?"**
- 🔴 Finding vulnerabilities before attackers do
- 🔵 Protecting systems and data from threats
- ⚪ Building secure systems from the ground up
- 🟣 Understanding how attacks work and tracking attackers
